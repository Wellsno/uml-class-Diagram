# UML Class Diagram Repository

## Instructions

This repository contains the UML Class Diagram for the ShopEase project.

### Files

* **ShopEase_Overall_Class_Diagram.svg** — View or download this file to see the complete UML Class Diagram.
* **diagram.puml** — Original PlantUML source code used to generate the diagram.

### How to View the Diagram

1. Open `ShopEase_Overall_Class_Diagram.svg`.
2. Download the SVG file if the preview is too small.
3. Open the SVG file in a web browser (Chrome, Edge, Firefox) for the best viewing experience.
4. Use browser zoom controls to inspect details.

### Additional Information

The `.puml` file is provided for reference and future modifications. The SVG file represents the final generated UML Class Diagram.

Prepared for academic project documentation purposes.
